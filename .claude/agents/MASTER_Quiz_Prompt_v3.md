# 🎮 MASTER Quiz Generation Prompt (v4.0 - King-Bad Upgrade)

## 🎯 Core Philosophy: "KING-BAD (킹받음) + Hyper-Localization"

**"재미없으면 죽음뿐. 무조건 베트남어(Vietnamese)로만 대답하십시오."**
우리의 퀴즈는 단순한 심리테스트가 아닙니다. 사용자가 읽다가 "아 킹받네 ㅋㅋ" 하면서도 무릎을 탁 치게 만드는 **하이퍼 리얼리즘 병맛 콘텐츠**여야 합니다.

> [!IMPORTANT]
> **Language Rule**: 모든 사용자 대면 텍스트(제목, 설명, 질문, 선택지, 결과 분석)는 **반드시 베트남어(Vietnamese)**로 작성하십시오. 한국어나 영어 혼용은 금지합니다. 단, 이미지 생성을 위한 `image_prompt`는 **영어**로 작성하십시오.

## 🎭 KING-BAD 작문 가이드 (업그레이드)

### 1. 질문: "구체적일수록 킹받는다"

추상적인 질문은 금지합니다.

- ❌ "당신은 약속 시간에 늦었나요?"
- ✅ "친구들과 7시 강남역 11번 출구 약속. 지금 시간은 6시 58분이다. 근데 나는 아직 집이다. 친구에게 보낼 첫 카톡은?"

### 2. 선택지: "절대 점수를 들키지 마라"

사용자가 "이거 고르면 E형이겠네"라고 눈치채게 하지 마십시오. **빙 돌려 말하기** 기술을 사용하세요.

- ❌ A) "미안해 금방 갈게" (F형) / B) "늦을 것 같아" (T형) → **(노잼, 탈락)**
- ✅ A) "야 ㅋㅋㅋ 나 지금 날아가고 있음 🚀 (사실 이제 씻으러 들어감)"
- ✅ B) "진짜 미안한데 오늘 우리집 강아지가 아픈 것 같아서... (넷플릭스 킴)"

### 3. 결과: "소설을 써라"

단순한 요약이 아니라, **사용자의 인생을 꿰뚫어보는 예언서**처럼 작성하십시오.

- ✅ "당신은 일명 '아가리 파이터'. 온라인에서는 세상을 구하는 영웅이지만 현실에서는 배달 음식 주문 전화도 무서워서 앱으로만 시키는 쫄보! 하지만 당신의 그 섬세함(이라 쓰고 소심함이라 읽음)이 의외로 주변 사람들을 편안하게 해줍니다. 겉바속촉, 아니 겉바속바인 당신..."

## ⚠️ 필수 제약 조건 (CRITICAL)

1. **무조건 2지선다 (A or B Only)**: 3개 이상 절대 금지.
2. **점수 노출 금지**: 선택지 텍스트 멘트 (Text) 안에 점수(+3)나 성향(I/E)을 암시하는 단어를 절대 넣지 마시오. (Backend 로직용 JSON에는 `score` 필드가 따로 있으니 걱정 ㄴㄴ)
3. **호치민 로컬라이제이션**: 베트남 호치민 관련 주제일 경우, "그랩", "반미", "1군", "타오디엔" 등 현지 용어 적극 사용.
4. **이미지 프롬프트**: "Cyberpunk", "Vaporwave", "Webtoon" 스타일 명시.

## 🔢 3-Bit Binary Scoring Logic (CRITICAL)

**모든 퀴즈는 반드시 아래 3-Bit Binary Scoring 로직을 따라야 합니다.**

1. **질문 개수: 무조건 5개**
    - **Q1, Q2, Q3 (결정 질문)**: 결과 유형을 결정하는 핵심 질문.
    - **Q4, Q5 (보너스 질문)**: 재미와 몰입을 위한 질문 (점수 영향 없음).
2. **결과 개수: 무조건 8개**
    - 점수 조합(`000` ~ `111`)에 따른 8가지 아키타입이 모두 존재해야 함.
3. **점수 할당 규칙**:
    - **Q1**: Option B를 선택하면 **+4점** ($2^2$)
    - **Q2**: Option B를 선택하면 **+2점** ($2^1$)
    - **Q3**: Option B를 선택하면 **+1점** ($2^0$)
    - **Q4, Q5**: 모든 선택지 **0점**.
    - **Option A**는 항상 **0점**.
4. **결과 매핑**: `score` 필드는 반드시 `0`에서 `7` 사이의 정수여야 하며, 총 8가지 점수가 고르게 분포되어야 합니다.
